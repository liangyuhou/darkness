Hypertext Markup Language(HTML)

The standard markup language for documents design to be displayed in a web browser. It can be assisted byCascading Style Sheets (CSS)and JavaScript.

Web browsers receive HTML documents from a web sever or from local storage and render the documents into mulyimedia web pages.
HTML describes the structure of a web page semantically and originallyincluded cues for the appearance of the document.

HTML elements are the building blocks of HTML pages.
With HTML constructs, images and other objects such as interactive forms may be embedded into the render page.
